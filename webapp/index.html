BrainFuck Interpreter
Brainfuck is a Turing complete language designed to be extremely minimalistic. It consists of only eight simple commands, a data pointer, and an instruction pointer. It is meant for amusement rather than practical function.
Command Description
>Move memory pointer to 1 unit right
<Move memory pointer to 1 unit left
+Increase memory at current location by 1
−Decrease memory at current location by 1
.Output the value at current location
,Input a value and store at current location
[ Like a while loop which runs while the memory at current location is not 0
] Closing of the loop. Loop back when value at current location is not 0
